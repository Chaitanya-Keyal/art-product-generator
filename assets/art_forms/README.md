# Art Form Reference Images

This directory contains reference images for each Indian art form.
The AI uses these images to understand and replicate the style.

## Image Loading

Reference images are **automatically discovered** at server startup.
Simply add images to the appropriate folder and restart the server.

## Instructions

1. Add 2-6 high-quality reference images per art form
2. Use any descriptive filename (e.g., `pattern1.jpg`, `traditional.png`)
3. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
4. Recommended size: 1024x1024 or larger
5. Choose images that clearly showcase the art style

## Best Practices

- Include variety: different subjects, compositions
- Ensure images are high resolution and clear
- Use authentic examples of the art form
- Avoid images with text overlays or watermarks
